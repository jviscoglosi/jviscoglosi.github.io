Make It or Break It

Users can set an overall wellness vision, log daily activities, set weekly goals, respond to weekly reflection prompts, and view their overall data, goals, and reflection responses. Natural Language Processor

Python

I used python to implement a neural network. This includes setting up initializing weights and bias, setting activation functions, cost functions, creating forward and backward propagation using gradient descent, etc. This network was used to classify vowel phonemes given an input of features of the position of the tongue, defined by the international phonetic alphabet. Results were back tested using PyTorch to verify the network's accuracy.

CORE: Scanner and Parser

Java

CORE is a made up language in which I wrote both a scanner and parser for in Java. The scanner went through and tokenized the program using lexical analysis. Next the parser would verify that the list of tokens were syntactically and semantically correct using an abstract syntax tree. This project helped me gain a deeper understanding of how source code gets processed by the compiler for a given language. The CORE language supported variables, functions, conditional statements, loops and even recursion.
